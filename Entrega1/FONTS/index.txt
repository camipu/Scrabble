- EXECUCIO DEL PROGRAMA
Per compilar:
1. Anar al directori fonts
2. ./grandlew build

Per provar el driver partida:
1. Anar al directori fonts
2. java -jar build/libs/DriverPartida.jar

Per provar el driver Estadistica:
1. Anar al directori fonts
2. java -jar build/libs/DriverEstadistica.jar

Per provar els tests
1. Anar al directori fonts
2. ./gradlew test
