Elements del directori:

- DOCS:
-- Conte la documentació del projecte

- EXE:
-- Executables del projecte

- FONTS:
-- Classes implementades
-- Tests unitàris implementats amb Junit4
-- Tests d'integració (Drivers)
-- Diccionari de paraules vàlides i fitxes (Resources)

Per compilar:
1. Anar al directori fonts
2. ./grandlew build

Per provar el driver partida:
1. Anar al directori fonts
2. java -jar build/libs/DriverPartida.jar

Per provar el driver Estadística:
1. Anar al directori fonts
2. java -jar build/libs/DriverEstadistica.jar

Per provar els tests
1. Anar al directori fonts
2. ./gradlew test
